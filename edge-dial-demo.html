How to Use

  • 1
    Click the green MENU button on the right edge to open the dial
  • 2
    Drag any icon up or down to rotate the dial and select different options
  • 3
    Click the active icon (the one at the edge) to trigger its action
  • 4
    Double-click the MENU button to enable slide mode (red border)
  • 5
    In slide mode, drag the MENU button up/down to reposition the entire dial
  • 6
    Notice the constraints: The dial won't slide beyond the top or bottom icon positions

üé® Design Philosophy

This component maintains the core interaction model from the original ang7.html canvas-based dial while modernizing it with web components, improved visuals, and enhanced functionality.

The half-dial design creates an elegant edge-mounted interface that doesn't obstruct content while remaining easily accessible. The center circle acts as both the anchor point and the primary interaction trigger.

‚öôÔ∏è Technical Implementation

The component uses a Web Component architecture with Shadow DOM for encapsulation. The dial maintains a centerY position that can be adjusted via the slide mode, with constraints ensuring it stays within radius + 50 from the top and bottom.

Icon positions are calculated using trigonometry, maintaining the same mathematical approach as the original: x = cos(angle) * radius and y = centerY + sin(angle) * radius.

The rotation system uses easing for smooth transitions, and the active icon detection uses angular distance calculations to determine which icon is closest to the 0-degree position (the right edge).
